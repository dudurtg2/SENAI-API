Sobre a SENAI API

Conheça nossa história, missão e como estamos transformando a educação através da tecnologia

Nossa História

A SENAI API nasceu da necessidade de criar uma ponte entre a academia e o mercado de trabalho, proporcionando uma plataforma onde alunos, professores e empresas possam se conectar através de projetos acadêmicos inovadores.

Desenvolvida com as mais modernas tecnologias e seguindo os melhores padrões de arquitetura de software, nossa plataforma representa um marco na modernização do ensino técnico e superior do SENAI.

Através desta ferramenta, estudantes podem não apenas documentar seus projetos, mas também showcasear suas habilidades para potenciais empregadores e investidores.

Inovação em Educação

Utilizamos tecnologia de ponta para criar soluções educacionais que preparam nossos alunos para os desafios do futuro.

Missão

Promover a inovação educacional através de uma plataforma digital que conecta estudantes, educadores e empresas, facilitando o compartilhamento de conhecimento e o desenvolvimento de projetos acadêmicos de impacto.

Visão

Ser a principal plataforma de projetos acadêmicos do SENAI, reconhecida pela qualidade, inovação e capacidade de transformar ideias estudantis em soluções reais para a indústria.

Valores

  • Inovação
  • Colaboração
  • Excelência
  • Transparência
  • Sustentabilidade

Arquitetura Técnica

Desenvolvida com as melhores práticas de engenharia de software

Clean Architecture

Nossa aplicação segue os princípios da Clean Architecture, implementando uma arquitetura híbrida que combina:

  • n-Layer Architecture: Separation of concerns clara
  • Hexagonal Architecture: Ports & Adapters para isolamento
  • Domain-Driven Design: Foco no domínio do negócio

Tecnologias

Backend

Java 17, Spring Boot 3.1.2, Spring Data JPA, Spring Security

Frontend

HTML5, CSS3, JavaScript, Thymeleaf

Database

PostgreSQL, H2 (desenvolvimento)

Principais Funcionalidades

Para Alunos
  • Cadastro e gerenciamento de projetos
  • Upload de anexos e documentação
  • Controle de visibilidade (público/privado)
  • Colaboração com outros estudantes
  • Showcase de habilidades
Para Professores
  • Orientação e avaliação de projetos
  • Acompanhamento do progresso
  • Feedback estruturado
  • Gestão de turmas e disciplinas
  • Relatórios de desempenho
Para Empresas
  • Visualização de projetos públicos
  • Identificação de talentos
  • Oportunidades de investimento
  • Conexão com desenvolvedores
  • Parcerias acadêmicas
Segurança
  • Autenticação OAuth2
  • Autorização baseada em roles
  • Criptografia de dados sensíveis
  • Auditoria de ações
  • Conformidade LGPD
